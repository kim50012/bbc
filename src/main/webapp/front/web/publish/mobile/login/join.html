对本条款中使用的用语定义如下。

1. “网上商城”是指公司为了给用户提供商品和服务，利用电脑等通信设备交易商品或服务的虚拟营业场所，同时代表网上商城的运营公司。

2. “用户”是指登陆网上商城并根据本条款接受“网上商城”所提供的服务的会员或非会员。

3. 会员是指通过向“网上商城”提供个人信息而完成注册，持续接受“网上商城”所提供的信息，并可以持续利用“网上商城”提供的服务的人。

4. “非会员”是指未进行会员注册而利用“网上商城”所提供的服务的人。

第 20 条（解决纠纷）

1. 对于用户之间发生的纠纷，网上商城应竭尽全力圆满解决纠纷。

2.“网上商城与用户之间发生的纠纷，可依据《电子文书以及电子商务交易基本法》32条，通过电子文书、电子交易纠纷解决委员会调解。

3.即使依据第2项规定也无法解决“网上商城”与用户之间发生的电子交易纠纷时，可向管辖区法院提起民事诉讼。

4. “网上商城”与用户之间提起的电子商务诉讼适用韩国法律第32条。
