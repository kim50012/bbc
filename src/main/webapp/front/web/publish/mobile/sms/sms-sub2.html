“新世界免税店 釜山店”是位于韩国代表性海边度假村釜山・海云台的购物圣地，在地下1层至地上3层的广阔空间内，陈列着琳琅满目的高档商品。位于3层的就餐区域，可以展望海云台全景，让您在购物之余乐享度假休闲的雅致。

“新世界免税店 釜山店”距离釜山地铁2号线海云台站步行12分钟，并且和位于海云台的高级度假村釜山乐园大酒店相连，同时每天都有多班从酒店及免税店至釜山市内的免费往返巴士，交通便利。临近免税店的海云台海水浴场，每当夏季韩国全境约有100万名游客聚集于此。秋季，这里会举办万众瞩目的“釜山国际电影节”，可以目睹众多亚洲巨星的风采。

免费往返巴士，游览海云台风光

“新世界免税店 釜山店”每天运营免费往返巴士，除了连接釜山市内酒店与免税店的西面路线、中央洞路线，自2013年又增设了游览海云台风光的海云台路线，该路线巴士穿梭于人气观光地迎月路、冬柏公园、海云台市场以及新世界百货店和大型超市易买得之间。免税店疯狂扫货之后，来一段海岸风光观赏，无比惬意。
